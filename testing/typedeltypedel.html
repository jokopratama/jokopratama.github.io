Pada contoh di atas, kita menggunakan dua elemen div dengan ID "typing-text" dan "additional-text" untuk menampilkan teks. CSS digunakan untuk mengatur tampilan elemen div. Dalam script JavaScript, kita menggunakan array texts yang berisi teks yang ingin ditampilkan. Fungsi typeText dan eraseText digunakan untuk mengatur efek pengetikan dan penghapusan teks secara berulang. Fungsi displayAdditionalText digunakan untuk menampilkan teks tambahan. Setelah menyimpan file tersebut, buka file "index.html" pada browser, dan Anda akan melihat teks pertama muncul dan menghilang, kemudian diikuti dengan teks tambahan. Anda dapat mengubah teks yang ingin ditampilkan dengan mengganti nilai array texts di dalam script JavaScript sesuai keinginan Anda. Selain itu, Anda juga dapat menyesuaikan tampilan teks dengan mengubah properti CSS pada elemen div dengan ID "typing-text" dan "additional-text". Semoga ini membantu!
